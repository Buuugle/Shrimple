# Shrimple [shader]

A Minecraft Java shader that attempts to maintain a minimal "vanilla" aesthetic, while adding some optional features:
 - FXAA (anti-aliasing)
 - Waving Plants
 - Soft Shadows
 - Colored Lighting \*
 - Ray Traced block lighting \*

\* Requires Iris 1.6 or later!
